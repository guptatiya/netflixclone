# 🎬 Netflix Landing Page UI Clone

A responsive Netflix Landing Page UI Clone built using **HTML5, CSS3, and JavaScript**. This project recreates the look and feel of the Netflix landing page with a modern, responsive design and interactive user interface. It is developed for learning purposes and to showcase front-end development skills.

## 🌐 Live Demo

🔗 https://guptatiya.github.io/netflixclone/

---

## ✨ Features

- 🎥 Netflix-inspired responsive UI
- 📱 Mobile-friendly layout
- 🧭 Responsive navigation bar
- 🎬 Trending Movies section
- 💡 "More Reasons to Join" feature cards
- ❓ Interactive FAQ accordion
- 📧 Email subscription form with validation
- 🎨 Smooth hover animations
- ⚡ Built with pure HTML, CSS, and JavaScript
